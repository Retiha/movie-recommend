A full-stack web application that allows users to browse, search, filter, and save their favorite movies. Built with **React** for the frontend and **Node.js + Express** for the backend.

## Features

- Browse a collection of movies with posters, genres, and ratings.
- Search movies by title.
- Filter movies by genre.
- Add or remove movies from your favorites.
- Separate favorites page to view saved movies.
- Responsive design for desktop and mobile.

## Tech Stack

- **Frontend:** React, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB (optional for favorites storage)
- **Others:** CORS, dotenv
